# Task 4.13: Add suppression counting and duplication metrics

## Metadata

- **Phase:** 4 — Metrics Integration
- **Task ID:** 4.13
- **Type:** tdd
- **Priority:** low
- **Depends on:** 4.01

## Description

Add two lightweight metrics collectors: (1) count inline suppressions (`# noqa`, `# nosec`, `# type: ignore`) across the codebase, and (2) extract duplication ratio from pylint similarities output.

## Acceptance Criteria

### Suppression Counting
- [ ] Count `# noqa` comments across source files
- [ ] Count `# nosec` comments across source files
- [ ] Count `# type: ignore` comments across source files
- [ ] Total suppression count
- [ ] Populates `metrics.suppressions`

### Duplication Metrics
- [ ] Extract duplicate block count from pylint similarities output
- [ ] Calculate duplication ratio (duplicate lines / total lines)
- [ ] Populates `metrics.duplication`

## Implementation Notes

- Suppression counting can be a simple grep/regex pass over source files — no AST needed
- Duplication data may already be available from pylint output; extract rather than re-analyze
- Both are lightweight additions, grouped into one task
